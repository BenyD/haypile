# Haypile — project rules

## Commits

- Write proper commit messages: simple, plain English, easy to understand at a glance.
- Subject line in imperative mood, under ~60 chars (e.g. "add FTS5 keyword search", "fix chunk overlap off-by-one").
- Add a short body only when the why isn't obvious from the subject.
- Never add a Claude trailer, Co-Authored-By, "Generated with", or any AI attribution to commits.
